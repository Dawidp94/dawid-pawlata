Hangman Game

This is a classic hangman game. The task is to guess the given password in up to 8 tries.

A game based on "clear" javascript.
50 randomly selected passwords implemented.
Graphics prepared by hand on figma.com.

Memory Game

An animal styled memory game. There are 6 pairs of cards to find, which are shuffled after each page reload. It was one of my first projects.

The site I use the jQuery library.

Weather App

Fully responsive app that shows the current weather for cities around the world. In addition, implemented: atmospheric pressure, wind strength, humidity and cloud cover.
